## What was tried

This is the **Wave 1** relentless-improvement push for tomorrow's 2026-06-10 大井 (Ōi, `keibajo_code=44`) NAR card, following the iter34 R5 similarity-member rejection ([`oi-2026-06-10-r5-nar-similarity-member.md`](oi-2026-06-10-r5-nar-similarity-member.md)). R5 closed by demanding a **genuinely-new signal that is BOTH orthogonal AND finish-predictive**. Wave 1 attacks that target on **five fronts in parallel**, each a distinct hypothesis about where unextracted finish signal might live, all judged on the **powered full-NAR per-class holdout** (n = 26060 C / 7217 other / 7124 B / 2812 A …) plus an Ōi(44) cross-check, under the strengthened gate (nested split + multi-metric ≥2/4 axes + ≥1 place positive + no axis < −0.05pp + top1 paired-bootstrap LB95>0 + Holm).

The five Wave 1 hypotheses:

- **H1 — field-relative / competitive-context features.** Within-race rank/zscore transforms of the existing absolute-level signals (market, speed, weight) as a competition-aware channel.
- **H2 — momentum / trajectory / freshness features.** Short-minus-long form deltas, recency slopes, freshness, weight change, consecutive-improvement counts — orthogonal-in-time signal distinct from absolute level.
- **H3 — calibration + meta-learner stacking.** Ridge / logistic / LGBM stack combiners and isotonic calibration over the existing per-class members.
- **H4 — HPO + alternative loss.** Fresh per-class Optuna hyperparameter search across multiple ranking-loss families (YetiRank / lightgbm:lambdarank / YetiRankPairwise / PairLogitPairwise / QuerySoftMax) on the per-class residual. **REJECTED all 3 classes** — verdict appended below.
- **H5 — multi-class probability head.** A single 7-class prob-head over all NAR classes. **Cut on memory** before producing a verdict (see below).

### Root-cause finding (the headline)

The **174-feature baseline is comprehensive**: it ALREADY contains every channel Wave 1 proposed — field-relative rank/diff cols, recency/days-since/form-trend/weight-diff cols, and within-race-normalized market signal. New features in those channels are **redundant**. The genuinely-orthogonal transforms (odds ordinal-rank, similarity geometry) carry **sub-noise** finish signal. **And H4 closes the capacity door:** the residual is **not under-capacity** — a fresh Optuna HPO with a better loss family (lightgbm:lambdarank) beat YetiRank on inner CV→WF for all 3 classes yet **failed the powered holdout gate**, only re-sharpening top1 at the cost of place ordering (exactly the iter13 JRA HPO CV→WF failure precedent). Combined with the NAR/Ōi holdout sizes, the available gains are **below the bootstrap-confirmable / place-preserving threshold**, so the strengthened gate **correctly rejects** them. **This is the accuracy CEILING for same-night feature / ensemble / HPO levers** over the current pipeline — neither features nor model fit moves it.
