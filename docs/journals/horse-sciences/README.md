# 馬の科学 (Uma no Kagaku) — JRA 競走馬総合研究所 機関誌コレクション

- **誌名**: 馬の科学 (Uma no Kagaku)
- **発行機関**: JRA 競走馬総合研究所 (Equine Research Institute, Japan Racing Association)
- **刊行範囲**: Vol.50 (2013年) – Vol.56 No.4 (2019年12月)。第56巻第4号をもって休刊。
- **出典 URL**: <https://company.jra.jp/equinst/publications/es.html>
- **備考**: JRA は Vol.50 については表紙・目次のみ PDF 公開（本文は Vol.51 以降のみ）。本文 PDF が公開されているのは Vol.51 以降。

## es.html 掲載文（原文）

「馬の科学」は、年4回総研が発行している馬に関する学術情報誌（非売品）で、 関係者の他、日本ウマ科学会会員を対象に配付していましたが、第56巻第4号（2019年12月）が最終号となります。

日本ウマ科学会につきましては、[日本ウマ科学会サイト](https://jses.jp/)をご覧ください。

PDFファイルをご覧いただくには、Acrobat Reader DCが必要です。Acrobat Reader DCをお持ちでない方は、バナーのリンク先から無料ダウンロードしてください。
